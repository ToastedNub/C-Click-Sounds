# C-Click-Sounds
 - Plays sound effects on mouse clicks and keyboard button presses

folders for presses and releases:
 - SOUNDS > MOUSE
 - SOUNDS > KEYBOARD
 - SOUNDS > SPACE
 - Each one has a RELEASE folder in it for releases

how it works:
 - You put .mp3, or .wav files in the folders (names do not rly matter)
 - Open the App
 - - Main will just play in your main audio output (headset etc)
 - - Mic will let you choose what mic output channel to play it in
 - Click one of the Buttons in the App to Enable / Disable them
 - Volume slider on the bottom

cleaning:
 - The Mic file can only use wav files
 - Run the Clean file to convert every file in the SOUNDS folder, and all its subfolders to wav files for you
 - - This will also move all your non wav files into the ARCHIVE folder
PYTHON IS REQUIRED FOR THIS PART
